# SupermanMusicPlayer

##简介
> 一款本地音乐播放器源码, 已实现的功能如下:

* 播放音乐
* 上一曲
* 下一曲
* 歌词和专辑图片的显示
* 添加歌曲到我的最爱
* 最近播放列表
* 专辑唱片列表
* 开启睡眠定时
* 铃音剪辑
* 第三方分享
